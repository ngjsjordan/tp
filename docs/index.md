**ClientSquare** is a one-stop **desktop application for freelance property agents** to help you **manage your client, appointments and listings efficiently**.

ClientSquare provides a **Command Line Interface (CLI)** for users who prefer typing while providing an intuitive **Graphical User Interface (GUI)** for the visual lovers.
